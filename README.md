# Cardano Savings Pool (Preprod)

## Overview
The Cardano Savings Pool is a decentralized savings platform built on Cardano (Plutus V2).  
Anyone can deposit ADA; only the owner can withdraw.  
Frontend uses Lucid with CIP-30 wallets (Eternl, Nami) on Preprod.

## Features
- Deposit ADA
- Owner-only withdraw
- Preprod network
- CIP-30 wallet support
